# 🎨 Calavero Design

Una skill de diseño para crear interfaces visualmente potentes con atención obsesiva a la jerarquía visual, contraste, espaciado, profundidad y micro-interacciones.

> **Diseñada para diseñadores y desarrolladores** que quieren que sus interfaces destaquen en un mar de componentes genéricos.

## 🚀 ¿Qué es esto?

Calavero Design es una **skill para GitHub Copilot** que codifica principios profesionales de diseño visual. Cuando la usas, Copilot puede:

✅ Revisar tu código UI y sugerir mejoras visuales específicas  
✅ Generar componentes siguiendo un sistema de diseño coherente  
✅ Aplicar jerarquía visual automáticamente  
✅ Asegurar contraste accesible (WCAG AA)  
✅ Implementar animaciones performantes con propósito  
✅ Crear sistemas de color, tipografía y espaciado consistentes

## 🎯 Principios Fundamentales

### 1. **El Contraste Visual es Jerarquía**
No uses boxes para separar, usa contraste de tamaño, peso, color y espacio.

### 2. **Los Detalles Invisibles Componen**
La suma de 100 decisiones correctas imperceptibles crea interfaces que la gente ama sin saber por qué.

### 3. **La Belleza es Ventaja Competitiva**
En un mundo de funcionalidad commodity, la experiencia visual es el diferenciador.

### 4. **Sistema sobre Casos Aislados**
Cada decisión visual debe ser replicable y escalable.

## 📚 Contenido de la Skill

- **Sistema de Tokens**: Colores, tipografía, espaciado, sombras, animaciones
- **Jerarquía Visual**: Cómo crear contraste efectivo
- **Profundidad**: Sistema de elevación con sombras
- **Animaciones**: Cuándo, cómo y qué animar
- **Accesibilidad**: Contraste WCAG, reduced motion
- **Componentes Base**: Button, Card, Input con ejemplos completos
- **Checklist de Revisión**: Para PRs con cambios visuales

## 🛠️ Tecnologías Recomendadas

Esta skill está optimizada para:
- **React** / **Next.js**
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (para animaciones avanzadas)

Pero los principios son universales y aplicables a cualquier stack.
